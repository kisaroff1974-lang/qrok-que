ПОДАЙ РЕПЛИКУ (GIVE CUE)
Версия: v1.0.0

------------------------------------
ОПИСАНИЕ ПРОЕКТА
------------------------------------

«Подай Реплику» — это приложение для актёров, режиссёров и студентов театральных и киношкол, 
предназначенное для работы с текстами сцен, самопроб, кастингов и репетиций.

Программа позволяет:
— загружать сценический текст
— автоматически разбивать его на реплики персонажей
— репетировать сцену в режиме «я / партнёр»
— воспроизводить записанные реплики партнёра
— повторять сцены столько раз, сколько необходимо

Проект создавался как инструмент живой актёрской практики, а не как чат-бот или нейросеть «ради эффекта».


------------------------------------
ОСНОВНЫЕ ВОЗМОЖНОСТИ
------------------------------------

• Создание новой сцены  
• Загрузка текста сцены (вставка текста, PDF, DOCX)  
• Автоматическое распознавание реплик по правилам сценарного формата:
  — имя персонажа написано ЗАГЛАВНЫМИ БУКВАМИ
  — после имени следует текст реплики  

• Пошаговая работа с репликами:
  — отображается имя персонажа и его текст
  — пользователь выбирает: «моя реплика» или «реплика партнёра»
  — для реплики партнёра можно записать голос

• Возможность переписать реплику после записи  
• Сохранение сцен и реплик в памяти приложения  

• Режим «Репетиция»:
  — реплики воспроизводятся по кнопке «Дальше»
  — текст всегда перед глазами
  — кнопки:
    • следующая реплика
    • предыдущая реплика
    • выход в меню

• Главное меню:
  — Новая сцена
  — Репетиция
  — Очистить сцену
  — Инструкция
  — Поддержать проект


------------------------------------
КАК ПОЛЬЗОВАТЬСЯ
------------------------------------

1. Нажмите «Новая сцена»
2. Вставьте текст сцены или загрузите файл
3. Программа разобьёт текст на реплики
4. Для каждой реплики выберите:
   — моя
   — партнёр
5. Для реплик партнёра запишите голос
6. После завершения сцены перейдите в «Репетицию»
7. Репетируйте, переключая реплики кнопкой «Дальше»

Приложение не ограничивает количество повторений и не навязывает темп.
